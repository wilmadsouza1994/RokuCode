#Roku Project Sample

The project model builds a video player channel. It walks developers through each step in developing a basic, high-performance Roku channel.
Its a reference when building a new Roku channel UI. Because this sample helps in understanding core concepts along with the flow,Its a template for quickly creating and publishing a customized Roku channel.

##Installation
To run the channels, follow these steps:

Download and then extract the channel.

In the extracted  the folder containing the channel you want to run and then compress the contents in the expanded folder to a ZIP file.

 Follow the steps in [Loading and Running Your Application](https://developer.roku.com/en-gb/docs/developer-program/getting-started/developer-setup.md
 #step-1-set-up-your-roku-device-to-enable-developer-settings) to enable developer mode on your device and sideload the ZIP file containing the sample onto it.
